Process Mining and System Dyanmics Solution


  • Use Event Log to SD Log for create SD Log including process parameters in an aggregated level.

  • Use TimeWindow Test for test the stability of the process parameters in the selected time window.

  • Use Relation Detection to find out about the correlation between parameters and setting threshold for their correlation.

  • Use Interactive CLD Graph Generation to set the relations between parameters based on their correlation and generate causal loop diagram.

  • Use Relation Detection Results to see the detailed relations between each parameters.

  • Use Generate SFD Graph for mapping the parameters onto the stock-flow elements and creating SFD.

  • Use Validation to check the level of agreement of values of parameters from event log with simulated values.
